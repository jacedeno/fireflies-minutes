## Trilium Integration

To save to Trilium, you can:

**Option 1: Copy/Paste**
1. Open GitHub `.md` file
2. Copy the content
3. Open Trilium
4. Create new note "2025-04-24 Maintenance Meeting"
5. Paste content
6. Add tags: `#meetings #maintenance`
7. Done!

**Option 2: Automatic (Future)**
If you add Trilium API integration to the routine, it can automatically:
1. Create note with same content
2. Auto-tag with meeting title
3. Link to GitHub source
4. Include backlink to GitHub
